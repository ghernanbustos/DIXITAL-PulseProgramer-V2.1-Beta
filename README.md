# DIXITAL-PulseProgramer-V2.1-Beta
This version has a complete pulse programmer; a sequence can be programmed by .txt instruction file. Reception is not ready to use, but there is a VHDL ADC driver block embedded in schematic, more architecture development  needs to be developed.

# Notebooks
- programSeq.txt contains pulse programer sequence code in machine language. This code execute a set of the operations to initialize chrono, ftw and ptw tunning words, Sinc modulation ... This file can be found in ...\DIXITAL-PulseProgramer-V2.1-Beta\HPS-DDS\Debug folder. Contains the last sequence code.
